# 猫叔 Skills 库

这是一个面向 Codex 的技能仓库，用来沉淀和复用猫叔设计的技能。

## 目录结构

- `skills-zh-cn/`：中文技能库
- `skills-en/`：英文技能库

## 当前技能

### `requirement-discovery`

用于把模糊需求、零散点子和多方分歧，通过多轮对话逐步收敛成可判断价值、可界定范围、可继续推进的需求结论。

### `skillforge-generator`

用于把重复工作、外部方法或已有流程，整理成可复用的 Codex 技能。

### `import-claude-skills`

用于扫描 Claude Code 风格的技能仓库，并将兼容技能迁移成 Codex 可用结构。

## 使用建议

- 中文场景优先使用 `skills-zh-cn/`
- 英文分享或英文环境优先使用 `skills-en/`
- 需要接入本机运行时，可从对应目录中挑选技能复制到实际 Codex `skills` 目录
