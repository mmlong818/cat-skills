# Import Claude Skills

This is the English companion for `import-claude-skills`.

## Purpose

Inspect Claude Code style skill repositories and import only the compatible skill content into Codex.

## Value

- discover candidate skills quickly
- import only useful material
- avoid copying Claude-specific assumptions blindly

## Recommended flow

1. prepare a local clone or extracted archive
2. scan for `SKILL.md`
3. inspect only relevant skills
4. check for Claude-specific dependencies
5. copy only core skill payload
6. adapt it into a Codex-native version
