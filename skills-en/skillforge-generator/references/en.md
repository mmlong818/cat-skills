# Skillforge Generator

This is the English companion for `skillforge-generator`.

## Purpose

Turn repeated work into a reusable Codex skill instead of leaving it as scattered notes or one-off prompts.

## Seven-step method

1. define the job in one sentence
2. collect realistic usage examples
3. separate reusable parts
4. strengthen the trigger description
5. write for execution
6. validate the shape
7. refine after real use

## Resource guidance

- `SKILL.md`: core workflow
- `scripts/`: repeatable deterministic actions
- `references/`: deeper supporting material
- `assets/`: templates and starter resources

## Signs of a good skill

- clear trigger conditions
- short operational instructions
- focused useful resources
- visible savings on future repeated work
